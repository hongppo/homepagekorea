# 홈페이지코리아 목업

홈페이지코리아의 화면 기획과 인터랙션 목업을 공유하기 위한 저장소입니다.

## 현재 목업

- AI 리드 분기 기반 문의 완료 페이지
- High 리드 상담 일정 선택 화면
- 관련 포트폴리오 노출
- PC·모바일 반응형 레이아웃

Google Calendar 영역은 검토용 인터랙티브 목업이며, 실제 예약 서비스와 전환 스크립트는 연결되어 있지 않습니다.

## 실행 방법

Node.js 22.13 이상이 필요합니다.

```bash
npm install
npm run dev
```

빌드 확인:

```bash
npm run build
```

## 주요 파일

- `app/page.tsx`: 완료 페이지 화면 및 일정 선택 인터랙션
- `app/globals.css`: 디자인과 반응형 스타일
- `app/layout.tsx`: 페이지 메타데이터 및 검색 제외 설정
